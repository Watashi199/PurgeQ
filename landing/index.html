// HOW IT WORKS

Three steps. No accounts. No telemetry.

// STEP
01
Run the server
One docker compose up. Postgres + Redis + the API on your machine, NAS, or a $5 VPS. The installer generates your API key.
// STEP
02
Load the extension
Grab the latest zip from Releases, drop it into chrome://extensions, paste your server URL and API key. Done in 60 seconds.
// STEP
03
Flag with one click
Hover any player card on FACEIT, hit Ban, type a reason. They'll be highlighted in red on every page from then on — for you and anyone sharing your server.
// FEATURES

Built to make the toxic invisible.

Visual highlight
Flagged players get a red glow + red nickname on every card. Hover to see the reason your past self left.
One-click flag
Hover, click Ban, type why. Your name is stamped automatically. Unban is one click too.
Live sync
New player joins your lobby? PurgeQ hits the server in real time. Your teammates' flags appear within seconds.
Team-shared lists
Point several extensions at the same server with the same key. Everyone sees the same list, edits propagate instantly.
Import / Export
Bulk-add from JSON or CSV. Export anytime for backup. Migrate from another tool, or seed your team with a starter list.
Works offline
The extension caches the last list locally. Highlights still show even if your server is asleep.
// PRIVACY

Your data, your machine.

PurgeQ is self-hosted by design. There is no PurgeQ Inc., no analytics, no account, no email collection. Just code you run.

Local-first
The banlist lives in your Postgres. The API runs in your network. Nothing leaves it unless you point your friends' extensions at the same URL.
No tracking
Zero analytics, zero telemetry, zero pings to a third party. The extension talks to your server and nothing else.
Open source
MIT licensed. Audit the 5k lines of TypeScript and Python yourself, or hand it to whoever you trust to.
No FACEIT account access
PurgeQ never reads your FACEIT credentials or match history. It just paints names on the pages you already see.
